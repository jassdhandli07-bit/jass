# Hospital Management System

A web-based application designed to manage hospital operations efficiently using PHP and MySQL.

## 🚀 Features
* **Doctor Management:** Add, edit, and delete doctor profiles.
* **Patient Records:** Maintain detailed patient history and reports.
* **Appointment Scheduling:** Book and track patient appointments.
* **Dashboard:** A central hub for viewing hospital statistics and quick navigation.
* **Billing System:** Generate and manage patient bills.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3
* **Backend:** PHP
* **Database:** MySQL

## 📂 Project Structure
* `add_doctor.php` - Interface to register new medical staff.
* `appointments.php` - View and manage scheduled visits.
* `db_connect.php` - Configuration for database connectivity.
* `hospital_db.sql` - The database schema for setting up the project.

## ⚙️ How to Run
1. Import the `hospital_db.sql` file into your local **phpMyAdmin**.
2. Update `db_connect.php` with your local database credentials.
3. Place the project folder in your `htdocs` (XAMPP) or `www` (WAMP) directory.
4. Access the project via `localhost/jass`.
